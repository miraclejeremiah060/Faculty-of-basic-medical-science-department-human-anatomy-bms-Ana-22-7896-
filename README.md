## What is Git?
Git is a distributed version control system used to track changes in source code over time.

### Functions of Git:
- Tracks changes in files
- Allows reverting to previous versions
- Supports collaboration
- Keeps record of project history using commits

---

## What is GitHub?
GitHub is a cloud-based platform that hosts Git repositories and allows collaboration among developers.

### How GitHub Works:
- Users create repositories
- Files are added and saved using commits
- Changes are pushed to GitHub servers
- Repositories can be shared publicly
